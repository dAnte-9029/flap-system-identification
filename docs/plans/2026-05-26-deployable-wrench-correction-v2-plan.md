## Model Forms

### Force Correction V2

Train deployable force correction:

```text
F_corr_v2 = F_prior + delta_F_v2(x, F_prior)
```

Use ridge linear correction with optional affine prior-force interactions:

```text
additive: delta_F = Phi(x) B
affine:   delta_F = Phi(x, F_prior * x) B
```

V2 intentionally starts with `additive` and `affine`; it does not include the old `multiplicative`
variant unless a validation-only result later motivates adding it.

### Moment Correction V2

Train only deployable moment corrections:

```text
direct_residual:
  M_corr = M_prior + delta_M(x)

force_arm:
  M_corr = r_hat(x) x F_corr_v2 + tau_free_hat(x)

hybrid:
  M_corr = M_prior + r_delta_hat(x) x F_corr_v2 + tau_free_hat(x)
```

Do not include any model whose `force_source == true_force` in final deployable comparisons.
